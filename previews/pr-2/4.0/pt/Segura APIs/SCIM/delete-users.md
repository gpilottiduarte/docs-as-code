# DELETE | Users

## Requisição
Para remover Usuários, faça uma solicitação ao seguinte endpoint:
`DELETE /scim/v2/Users/:id`

## Parâmetros
Sem parâmetros.

## Retorno
Se a exclusão do usuário for bem-sucedida, a resposta 'HTTP 204 No Content' será retornada.

### Exemplo de resposta
`HTTP/1.1 204 No Content`
