# Domum health check

This document provides information about the **Domum health check** screen in the **Audit** menu, which displays details of **Domum**â€™s connection and allows you to check if the component is functioning properly.

## Path to access

1. On Segura, in the navigation bar, hover over the **Products menu** and select **Domum Remote Access**.  
2. In the side menu, select **Audit \> Domum health check**.

## Actions menu

| Item | Type | Description |
| ----- | ----- | ----- |
| **Actions** | Dropdown menu | Displays the options *Print report*, *Export CSV*, and *Schedule report*. |

## Search fields

| Item | Type | Description |
| ----- | ----- | ----- |
| **Date/Time** | Date picker | Filters the connections by the date and time. |
| **Error** | Dropdown menu | Filters the connections by the presence or absence of an error in Segura's communication with the **Domum** gateway. |

### Report fields

- **Date/Time**.  
- **Error**.  
- **Details**: details of the connection.

:::(info) (Info)  
By default, the report displays 30 records per screen. To go to the next screen, click the forward buttons at the end of the report.  
:::
